# 🔐 SIEM Log Analyzer

> **Skills demonstrated:** SIEM platforms (Splunk/Sentinel logic) · Threat detection · Incident Response · MITRE ATT&CK mapping · NIST 800-53 / ISO 27001 / ASD Essential Eight compliance

A production-grade, multi-format Security Information and Event Management (SIEM) log analysis engine built in pure Python. Designed to mirror the core logic used in enterprise tools like **Splunk**, **Microsoft Sentinel**, and **IBM QRadar** — 
---

## 📋 Why This Project

| Skill | Where Used in This Project |
|---|---|
| SIEM platforms (Splunk / Sentinel) | Core correlation engine mirrors Splunk SPL logic |
| Threat detection & analysis | 5 detection rules with sliding-window correlation |
| Incident Response | Alert triage scoring, recommended actions |
| MITRE ATT&CK knowledge | Every alert maps to ATT&CK technique IDs |
| Compliance (NIST, ISO, Essential Eight) | All alerts include control references |
| Python scripting | Entire codebase |
| Log parsing (Syslog, Apache, CloudTrail) | Multi-format parser with regex + JSON |

## 🚀 Quick Start

### Prerequisites
- Python 3.11+
- No external dependencies (stdlib only)

```bash
git clone https://github.com/F45elix/siem-log-analyzer.git
cd siem-log-analyzer

# Run full demo (generates synthetic logs + full report)
python main.py --demo

# Analyse your own log file
python main.py --log /var/log/auth.log --format syslog --report reports/

# Run tests
pip install pytest
pytest tests/ -v
```
